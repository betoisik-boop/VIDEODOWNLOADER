# Video Downloader

YouTube (MP4 / MP3) ve Pinterest tek-link indirici. Tkinter GUI.

## Windows kullanıcıları

[Releases sayfasından](../../releases) en son `VideoDownloader.exe` dosyasını indir ve çalıştır. Kurulum yok.

## Build (otomatik)

`main` branch'e her push'ta GitHub Actions Windows runner'da PyInstaller ile
tek-dosya `.exe` üretir ve Releases'a yükler.
